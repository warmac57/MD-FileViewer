## Additional Information

This viewer uses the **CDS.Markdown** package to convert Markdown to HTML, which is then rendered in a WebBrowser control. The styling is applied via CSS to create a clean, readable experience.

### Keyboard Shortcuts

- **Ctrl+O**: Open file
- **F5**: Reload
- **Ctrl++**: Increase font
- **Ctrl+-**: Decrease font
- **Ctrl+0**: Reset font
